Budget Wiz

Demo video Live site

Budget Wiz is an online budgeting application for creating and tracking budgets. It gives you the ability to create a budget, add budget categories, and track expenses and incomes. All data is stored in a Heroku postgreSQL database, so it will persist over time. This project utilizes a login system with authentication. Budget Wiz is built with vanilla Javascript, and uses many tools/libraries such as Jest, bcrypt, axios, express, SQL, postgres, sequel, bcrypt, etc.
